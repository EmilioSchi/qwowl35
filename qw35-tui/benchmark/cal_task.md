Implement a program that mimics the basic behavior of the Unix cal command.

Before implementing it, run the cal command in a Bash shell to understand the expected output format. Do not support the many command-line options of the real cal utility; only implement the simplest use case (displaying the current month's calendar).

The program should print a calendar page with the same layout and character alignment as the standard cal output. Pay special attention to spacing and column positions, since the visual formatting is important.

To verify correctness, write automated tests that compare the generated output against an expected string. The tests should perform an exact string comparison, including whitespace and line breaks.
